# apiClientes

Para visualizar a documentacao da api, acesse: http://localhost:8080/swagger-ui/index.html

## Tecnologias utilizadas
- Java 17
- Gradle
- Postman
- Swagger / Spring Open Api
- Spring
- Bean Validation
- H2 Database
- JUnit

## Execução
### Requisitos
- Java 17
- Gradle

### Como rodar
- Baixar o projeto para uma pasta local no computador
- Abrir o projeto em uma IDE de desenvolvimento
- Baixar as dependências do projeto, limpar e construir o projeto usando o Gradle
- Executar o projeto. O Spring se encarregará de subir o servidor Tomcat e fazer o deploy da aplicação no mesmo, deixando a aplicação disponível para acesso através da URL: http://localhost:8080/clientes
